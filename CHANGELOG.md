| v0.28.0 | 2026-06-03 | A2A Mesh L3a: cross-Bridge pipeline + S3 workspace relay. S3 is a HARD prerequisite: without it a cross-node step FAILS with a clear error (never silently loses files); local-only pipelines never touch S3. Only A needs S3 write; B uses presigned URLs (no creds). 1-hop limit + local priority carried from L2. L3b (multi cross-node step chains, parallel-mode workspace merge, incremental diff) deferred. |
